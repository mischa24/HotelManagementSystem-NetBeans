Φτιάξε σε Java μια εφαρμογή διαχείρισης κρατήσεων για ένα μικρό ξενοδοχείο.
Η εφαρμογή επιτρέπει την καταχώριση στοιχείων κρατήσεων πελατών και 
υπολογίζει τους λογαριασμούς τους με έκπτωση και ΦΠΑ.

✅ Λειτουργίες που απαιτούνται
Ο χρήστης δίνει πόσες κρατήσεις θέλει να εισάγει.
Για κάθε κράτηση καταχωρεί:

Όνομα πελάτη

Αριθμό δωματίου

Αριθμό διανυκτερεύσεων

Τιμή ανά διανυκτέρευση

Επιπλέον χρεώσεις

Το πρόγραμμα αποθηκεύει όλες τις κρατήσεις σε μονοδιάστατο πίνακα αντικειμένων.

Για κάθε κράτηση υπολογίζεται:

Υποσύνολο = (διανυκτερεύσεις * τιμή ανά διανυκτέρευση) + επιπλέον χρεώσεις

Αν οι διανυκτερεύσεις > 5 → έκπτωση 10%

ΦΠΑ 13% στο ποσό μετά την έκπτωση

Τελικό ποσό = έκπτωση + ΦΠΑ

Το πρόγραμμα εμφανίζει αναλυτική απόδειξη για κάθε πελάτη:

Όνομα

Δωμάτιο

Διανυκτερεύσεις

Τιμή ανά διανυκτέρευση

Επιπλέον χρεώσεις

Υποσύνολο

Ποσό μετά την έκπτωση

ΦΠΑ

Τελικό ποσό

Στο τέλος εμφανίζει:

Μέσο όρο τελικών ποσών όλων των πελατών

Ποιος πλήρωσε το υψηλότερο ποσό
